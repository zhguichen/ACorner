# PRD：一隅 ACorner

## 问题与目标

持续可见的倒计时、效率评价和强提醒会打断本已进入状态的用户。一隅只帮助用户为当前一件事留出一段时间：时间存在，但不持续催促。

目标用户是希望在 macOS 桌面上安静推进一件事、又不希望使用完整任务管理工具的人。

成功标准是用户可以从灰色圆点开始一项任务，在不看秒数的情况下自然结束或转入下一项；完成记录能够可靠地保存在自己选择的本地文件夹，重启后不会丢失正在进行的任务或草稿。

## 核心流程

1. 启动时只显示可拖动的灰色圆点。
2. 展开卡片，输入目标并以滚轮选择投入分钟数。
3. 开始后圆点变绿，默认不展示任务和时间。
4. 到计划结束时圆点变黄并展开；十秒未操作则变蓝并进入正计时。
5. 用户可完成任务并填写自由备注，或直接开始下一件事。
6. 首次写入时选择本地文件夹；之后自动更新该文件夹中的记录。

## 状态与文案

| 状态 | 圆点 | 对用户的文字 |
| --- | --- | --- |
| 空闲 | 灰色 | 空闲 / 现在想推进什么？ |
| 计划内 | 绿色 | 正在进行 / 剩余约 N 分钟 |
| 到时等待 | 黄色 | 这段时间结束了 |
| 正计时 | 蓝色 | 仍在投入 / 已继续约 N 分钟 |

颜色不是唯一的信息表达。时间只显示模糊分钟数，绝不显示秒数或“超时”类失败表述。

## 明确不做

不做暂停、番茄钟循环、休息提醒、声音、强制通知、账号、云同步、跨设备同步、AI、屏幕监控、效率评分、排行榜、复杂统计、项目/标签/优先级或多任务并行。
