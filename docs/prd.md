# PRD：一隅 ACorner

## 问题与目标

持续可见的倒计时、效率评价和强提醒会打断本已进入状态的用户。一隅只帮助用户为当前一件事留出一段时间：时间存在，但不持续催促。

目标用户是希望在 macOS 桌面上安静推进一件事、又不希望使用完整任务管理工具的人。

成功标准是用户可以从灰色细胶囊条开始一项任务，在不看秒数的情况下自然结束或转入下一项；计时中和结束后都能在同一个记录框里写备注；胶囊条内的进度线会随着待办完成度逐步填满；完成记录与待办清单都能可靠地保存在自己选择的本地文件夹，重启后不会丢失正在进行的任务、记录草稿或标题草稿。

## 核心流程

1. 启动时只显示可拖动的灰色细胶囊条，胶囊条内含状态文字和待办完成度线。
2. 展开面板，输入目标并键入 1 到 180 分钟的投入时长。
3. 用户可从空闲卡片的“待办”按钮打开待办窗口，维护待办与已完成事项；完成项以勾选状态展示。
4. 开始后长条变绿，默认不展示任务和时间；展开后可以在较大的记录框里输入过程记录。
5. 到计划结束时长条变黄并展开；十秒未操作则变蓝并进入正计时。
6. 用户点击“结束”后进入收尾状态，继续编辑同一个记录框。
7. 用户点击“Next Mission”后保存记录、勾选关联待办，并回到下一任务输入。
8. 首次写入时选择本地文件夹；之后自动更新该文件夹中的任务记录与待办清单，设置窗口可查看待办完成项和普通完成任务。

## 状态与文案

| 状态 | 长条 | 对用户的文字 |
| --- | --- | --- |
| 空闲 | 灰色 | 空闲 / 现在想推进什么？ |
| 计划内 | 绿色 | 正在进行 / 剩余约 N 分钟 |
| 到时等待 | 黄色 | 这段时间结束了 |
| 正计时 | 蓝色 | 仍在投入 / 已继续约 N 分钟 |

颜色不是唯一的信息表达。长条始终同时显示状态文字；时间只显示模糊分钟数，绝不显示秒数或“超时”类失败表述。

## 明确不做

不做暂停、番茄钟循环、休息提醒、声音、强制通知、账号、云同步、跨设备同步、AI、屏幕监控、效率评分、排行榜、复杂统计、项目/标签/优先级或多任务并行。
