# 一隅 ACorner

屏幕一隅，专注一件事。

一隅是一个原生 macOS 悬浮计时器。它用一个安静的圆点陪伴当前任务：不持续显示秒数、不播放提示音、不评价效率；只有在你主动展开时才展示任务和大致时间。

## 快速开始

需要 macOS 14+，并已安装 Xcode 或 Xcode Command Line Tools。进入项目根目录后运行：

```sh
swift build
swift run
```

启动后只会出现灰色圆点。悬停或单击圆点可展开任务卡片；展开后的卡片会保留，点击卡片外部才会收起。输入任务、用滚轮选择投入分钟数并点击“开始”即可计时。

首次完成或切换任务时，应用会请你选择一个本地文件夹，并在其中维护 `ACornerTasks.json`。圆点位置、未开始草稿和未结束任务状态保存在本机偏好设置中。

## 开发

```sh
swift build
swift test
swift run
```

项目采用 Swift Package Manager；详细的人工验收步骤见 [docs/test.md](docs/test.md)。产品、架构和迭代约定见 [docs/](docs/)；给后续 AI 编程助手的修改规则在 [AGENTS.md](AGENTS.md)。

## 范围

当前版本负责单任务计时、自然进入正计时、自由备注、本地记录和重启恢复。

当前版本不提供暂停、番茄钟循环、强提醒、云同步、账号、统计看板、任务管理、屏幕监控或 AI 功能。

## 已知限制

- 当前以 Swift Package 形式运行，尚未提供签名和公证后的 `.app` 安装包。
- 任务历史仅保存为本地 JSON 文件，应用内暂不提供浏览或统计界面。
- 最低系统版本是 macOS 14。
